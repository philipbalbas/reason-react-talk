## Syntax Cheatsheet

| JavaScript            | Reason                |
| --------------------- | --------------------- |
| `var str = 'string'`  | `let str = "string"`  |
| `let num = 1`         | `let num = 1`         |
| `const arr = [1,2,3]` | `let arr = [|1,2,3|]` |
| `"hello " + "world"`  | `"hello " ++ "world"` |
| `1 + 1`               | `1 + 1`               |
| `1.2 + 2`             | `1.2 +. 2.`           |
| `const {a, b} = data` | `let {a, b} = data`   |
| `null`, `undefined`   | `None`                |
